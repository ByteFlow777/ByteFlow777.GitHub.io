      echo 服务器 #

      流程 #

      服务端（被动）接受连接请求的套接字创建过程如下：

      1. 调用 socket 函数创建套接字。
      2. 调用 bind 函数分配 IP 地址和端口号。
      3. 调用 listen 函数转为可接受请求状态。
      4. 调用 accept 函数处理连接请求。

      客户端（主动）连接请求的套接字创建过程如下：

      1. 调用 socket 函数创建套接字。
      2. 调用 connect 函数与服务器建立连接。
